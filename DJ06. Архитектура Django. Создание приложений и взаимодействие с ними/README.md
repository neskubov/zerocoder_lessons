# zerocoder_lessons

Повтори все действия за преподавателем, чтобы закрепить работу с кодом и инструментами, разобранными в уроке.
Пройди тест в конце занятия, чтобы проверить свое понимание ключевых концепций.
Отметь галочками выполненные шаги ниже.
Данное задание принимается автоматически после заполнения всех галочек и успешного прохождения теста.

Создан Django-проект с помощью команды django-admin startproject mysite, и код организован в соответствующей структуре. - * обязательное поле
В проекте созданы отдельные приложения через django-admin startapp, каждое отвечает за свою логику и не дублирует код других. - * обязательное поле
Приложения добавлены в INSTALLED_APPS в settings.py, корректно настроены urls.py и views.py для маршрутизации. - * обязательное поле
В коде отсутствуют циклические импорты между приложениями, а зависимости организованы через get_user_model() или импорт внутри методов. - * обязательное поле
API-ключи и конфиденциальные данные хранятся в .env файле, подключенном через dotenv, а не в коде settings.py. - * обязательное поле
